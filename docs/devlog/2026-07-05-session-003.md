# SHAI Devlog - 2026-07-05 - Session 003

## Summary

Added post-export STL mesh validation, lightweight geometry/STL regression coverage, and an optional vector extrusion STL backend that falls back to the existing raster height-field algorithm.

## Why

SHAI needs to know whether an STL is merely present or actually sane enough to inspect, upload, and eventually print. A structured mesh report gives CLI, GUI, tests, and future upload packaging a shared source of truth.

## Files Changed

- `spool_house_ai/processing/stl.py`: Added `MeshReport`, STL validation, and JSON report writing helpers.
- `spool_house_ai/processing/geometry.py`: Replaced deprecated 2D `np.cross` usage with the equivalent scalar cross-product distance calculation.
- `spool_house_ai/pipeline.py`: Runs validation after STL export, saves `mesh_report.json`, logs warnings/failures, and records backend fallback behavior.
- `spool_house_ai/config.py`: Adds the `stl_backend` config field.
- `config/config.yaml`: Defaults `stl_backend` to `raster_heightfield`.
- `spool_house_ai/main.py`: Adds the `--stl-backend` CLI override.
- `spool_house_ai/gui.py`: Adds a simple STL backend selector in the settings panel.
- `spool_house_ai/test_mode.py`: Verifies generated mesh reports exist and contain sane mesh stats.
- `tests/test_geometry_regression.py`: Adds deterministic regression coverage for holes, details, island removal, geometry drift, and STL mesh stats.
- `tests/__init__.py`: Marks the test folder as an importable package.
- `.github/workflows/ci.yml`: Runs standard-library unittest discovery before pipeline test mode.
- `README.md`: Documents mesh reports and STL backend selection.
- `docs/devlog/2026-07-05-session-003.md`: Records this development checkpoint.

## Features Added

- Added per-job `mesh_report.json` files with STL file size, vertex count, face count, bounds, empty mesh detection, invalid bounds detection, warnings, and failures.
- Added a focused regression test for geometry mask classification and STL report sanity.
- Added optional `vector_extrusion` STL backend selection with `raster_heightfield` as the safe default fallback.
